## INSTALL   
Can be installed through the official Kodi repository.
To install this add-on you must do so from the Add-on manager from within Kodi by going to:   

1. Settings   
2. Add-ons   
3. Install from repository   
4. Kodi add-on repository   
5. Look and feel   
6. Skins   
7. AppTV   
8. Install
